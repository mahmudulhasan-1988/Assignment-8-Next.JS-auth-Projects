# 🧱 Hasan Tiles Gallery

## 📌 Project Overview

Hasan Tiles Gallery is a modern web application built with Next.js that showcases a collection of premium ceramic and decorative tiles. Users can browse different tile categories, view detailed product information, and explore stylish designs for homes and commercial spaces.

---

## 🎯 Purpose

The purpose of this project is to:

* Provide a clean and user-friendly tile browsing experience
* Display product details in an organized way
* Practice modern frontend development using Next.js and React
* Build a real-world e-commerce style UI

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/hasan-tiles-gallery.git
```

Go to project folder:

```bash
cd hasan-tiles-gallery
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
